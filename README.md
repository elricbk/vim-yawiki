# vim-yawiki

Плагин для работы с вики-разметкой. Поддерживает:
* Подсветку синтаксиса
* Folding секций и cut-ов (`<{` и `}>`)
* Переход по заголовкам с помощью `]]` и `[[`
* Биндинги для форматирования текста
